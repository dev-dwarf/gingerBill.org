Memory Allocation Strategies - Part 1

2018-02-01

Ginger Bill

ROUGH DRAFT

Memory allocation seems to be something many people hate to do or are bad at. Many languages have automatic memory management

Allocation Qualities

  • Size
  • Lifetime
  • Usage (performance) (secondary concern)

Size-Lifetime Chart

Generations of Lifetimes

Permanent Allocation

Never Free - Persistent during program lifetime

Transient Allocation

Cycle-based lifetime (free per frame, loop, etc)

Scratch/Temporary Allocation

Short lived, quick memory

Out of Order Allocation

General purpose memory that requires lifetimes < program lifetime
